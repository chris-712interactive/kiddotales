# Signature fonts for electronic W-9

To enable **Handwriting** and **Cursive** signature options, add these font files here:

| Option     | File name           | Download |
|-----------|---------------------|----------|
| Handwriting | `caveat.ttf`      | [Caveat on Google Fonts](https://fonts.google.com/specimen/Caveat) → Download family → extract `Caveat-Regular.ttf` and rename to `caveat.ttf` |
| Cursive   | `dancing-script.ttf` | [Dancing Script on Google Fonts](https://fonts.google.com/specimen/Dancing+Script) → Download family → extract `DancingScript-Regular.ttf` and rename to `dancing-script.ttf` |

Place the `.ttf` files in this folder. The app will load them for the signature font picker and PDF generation.
